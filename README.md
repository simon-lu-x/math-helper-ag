# 湖南老教师数学课件助手 (Math Helper AG)

> 让每一份手写的智慧，都能转化为璀璨的数字课件。

本工具专为湖南地区资深数学教师打造，旨在通过 Gemini 2.5 Flash 的强大能力，将传统教案手稿一键转化为排版标准、公式精美的电子文档。

## 🌟 核心特色

- **多图识别**：支持连续拍摄多张教案纸，AI 自动串联逻辑，生成完整课件。
- **岳阳方言助手**：深度优化针对岳阳方言的识别逻辑，老师动动嘴，AI 动手动笔。
- **高对比度适老化设计**：采用大字体、高对比度配色和清晰的交互逻辑。
- **一键导出**：支持 PDF 和微信长图，兼容教室投影和 U 盘存储。

## 🚀 快速启动

```bash
npm install
npm run dev
```

## 🛠 技术底层

- **模型**：Gemini 2.5 Flash (OCR + Math + Dialect Interpretation)
- **前端**：React + Tailwind CSS 4 + Framer Motion
- **公式**：KaTeX + Remark Math
- **导出**：html2canvas + jsPDF

## 👨‍🏫 设计初衷

资深教师拥有丰富的教学经验，手写稿往往承载着独特的教学艺术。我们用技术保留这份温度，用 AI 消除数字鸿沟。
